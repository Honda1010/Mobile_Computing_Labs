# Mobile Development

A monorepo of mobile projects built with React Native / Expo. Each top-level directory is a standalone project.

## Projects

| Directory | Description |
|-----------|-------------|
| [MyTodoList](./MyTodoList) | A simple Todo List app built with Expo & React Native |

## Getting Started

Each project is self-contained. Navigate into the project folder and follow its own setup steps:

```bash
cd <project-name>
npm install
npx expo start
```

## Structure

```
Mobile_Development/
├── MyTodoList/          # Todo List app (Expo)
└── ...                  # Future projects
```
